## Requirements

- Create a Chrome MV3 extension scaffold and build pipeline.
- Add configurable target language settings persisted in extension storage.
- Inject a manual Translate action on detected LinkedIn carousel posts.
- Capture the visible carousel slide and send it for OCR and local translation.
- Render translated output in a draggable in-page widget.

## Implementation Plan

1. Add manifest, build tooling, and extension entrypoints.
2. Implement popup and options for target language configuration.
3. Implement content script for DOM observation, button injection, and widget rendering.
4. Implement background and offscreen processing for OCR plus translation.
5. Validate with build checks and extension loading steps.

## Verification

- npm install completes successfully.
- npm run build outputs a loadable extension package.
- Opening LinkedIn shows Translate controls on carousel posts.
- Triggering translation returns localized text in the in-page widget.
